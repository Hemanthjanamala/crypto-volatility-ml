# Crypto Volatility ML
Forecasting and analysing cryptocurrency volatility with ARIMA, LightGBM, and LSTM.

## Quickstart
1) Create venv, install requirements: `python -m venv venv` → activate → `pip install -r requirements.txt`
2) Put data in `data/raw/crypto_all_combined.csv`
3) Run notebooks: `notebooks/00_eda.ipynb` → `01_models.ipynb` → `02_results.ipynb`
